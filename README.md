# Family Hub

App React + Vite per archivio famiglia, salute e viaggi.

## Avvio locale

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy GitHub Pages

Il repository è configurato per il deploy automatico con GitHub Actions.

1. Push su `main`
2. Vai in **Settings > Pages**
3. Seleziona **Build and deployment = GitHub Actions**

L'app sarà pubblicata su:

`https://crimifrancesco-art.github.io/family-hub/`
